BATTLE OF NAMES
RULES

名前から戦士を召喚し、作戦をロックして、オートバトルで勝負します。

流れ

  1. 名前を入れて SUMMON
  2. TACTICS で作戦を入力
  3. BATTLE START で対戦開始

作戦

作戦特徴
↑ 速攻序盤に手数で押す。長引くとバテる。
→ 攻撃一撃と必殺が重い。少し遅く打たれ弱い。
↓ 防御序盤の攻撃を受け切る。速攻に強い。
← 覚醒HP半分で回復し、必殺率アップ。連続必殺の夢。

4すくみ：速攻 → 覚醒 → 攻撃 → 防御 → 速攻
色分け：1P=青、2P=赤。作戦色：速攻=青、攻撃=赤、防御=黄、覚醒=紫。POWとステータス内HPを削除し、二つ名と ATK / DEF / SPD / LUK を大きく表示します。二つ名は真面目系とネタ系が混ざります。
バトル中は中央の実況テロップがメイン、下部ログは履歴表示です。

操作

作戦はキャラ周囲の4ボタンをクリック、またはキャラをクリックしてから W/A/S/D・矢印キー・コントローラー十字キーでロックします。スマホではキャラ周囲を上下左右にフリックして作戦をロックできます。選択後はLOCKED表示、バトル開始時に公開されます。
